# 家庭提醒系統

基於 Node.js 的雲端家庭提醒服務，支援定時推播與同步功能。

[![Deploy to Render](https://img.shields.io/badge/Deploy%20to-Render-46E3B7?style=for-the-badge&logo=render&logoColor=white)](https://render.com/deploy?repo=https://github.com/ken851004-afk/family-reminder-cloud)

點擊上方按鈕即可一鍵部署至 Render，享受 24/7 全天候運行的提醒服務。
